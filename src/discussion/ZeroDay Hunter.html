ZeroDayHunter

A Machine Learning-Based Malware Detection System

ZeroDayHunter is a group senior project designed to detect malicious and benign files using machine learning models such as Random Forest and XGBoost.

Project Overview

ZeroDayHunter aims to improve malware detection by using machine learning techniques instead of relying only on traditional signature-based detection. The system analyzes file features and classifies samples as malicious or benign.

Problem Statement

Traditional malware detection methods struggle to detect unknown, zero-day, and polymorphic malware. This project explores machine learning as a more adaptive approach for identifying threats based on extracted features and learned patterns.

How It Works

1. Data Processing

The system loads data from the EMBER dataset, extracts features, handles missing values, and prepares the data for machine learning.

2. Model Training

Machine learning models such as Random Forest and XGBoost are trained using processed training data.

3. Evaluation

The models are evaluated using accuracy, ROC-AUC, classification reports, and confusion matrices.
